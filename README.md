# 🛒 E-Commerce Sales Analysis (Olist Dataset)

## 📌 Overview

This project performs an in-depth exploratory data analysis (EDA) on a real-world Brazilian e-commerce dataset to uncover key business insights such as revenue trends, customer behavior, and order patterns. The analysis is conducted using Python and Pandas, with visualizations to support data-driven conclusions.

---

## 🎯 Objectives

* Analyze overall sales performance and revenue generation
* Identify monthly and seasonal sales trends
* Examine order statuses and customer purchasing behavior
* Derive actionable insights for business decision-making

---

## 📊 Dataset

* Source: Kaggle
* Dataset: Brazilian E-Commerce Public Dataset (Olist)
* Contains information on orders, payments, customers, and products

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib

---

## 🔍 Key Analysis Performed

* Data cleaning and preprocessing
* Merging multiple datasets for comprehensive analysis
* Revenue calculation from transaction data
* Monthly sales trend analysis
* Order status distribution

---

## 📈 Key Insights

* Identified peak sales periods and revenue fluctuations over time
* Observed distribution of order statuses (delivered, canceled, etc.)
* Highlighted patterns in customer purchasing behavior
